Welcome to the (unofficial) State osu Leaderboard!

The intention of this site is to allow users to compare themselves on a more local level, as well as connect players that live near each other!

A "✓" next to a user's state means they added themself to the website and confirmed their state selection.

A "✓" next to a user's discord tag means they connected their discord account using the "-link [osuid]" command through botbar.

If a state has a discord server, it will appear in the top bar on that state's leaderboard page. If there isn't one, try asking one of the players in that state if it is invite only/needs to be created.
