How does this work?

To add yourself to the list or update your stats enter your account ID and your state then hit submit!

All added users are updated every night around 4am EST, this takes around 10-30 minutes. During this time all users being added will be deleted. This feature is still experimental so expect some bugs for the time being. Once in the database, your user data is added to the appropriate state list. If you want a state discord server added to your state's page, please contact me at the link above!

If you currently associate with more than one state, for now pick the one you prefer. I did not create the site with that in mind, so I do not currently support it. I'll try and add that functionality in the future!
